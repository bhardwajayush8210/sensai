# 🚀 SensAI — AI-Powered Student Preparation Platform

🎯 Built with a modern full-stack architecture focusing on **performance, scalability, and real-world usability**.

## ⚙️ How It Works

1. User signs up via Clerk authentication
2. Access personalized dashboard
3. Explore **industry insights (bar graphs)**
4. Attempt quizzes and track performance
5. Input details (skills, education, etc.)
6. AI generates:

   * Resume
   * Cover Letter
7. Background processes handled via Inngest

## 🛠️ Local Development

### 1. Clone the repository

```bash
git clone https://github.com/your-username/sensai.git
cd sensai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env` file:

```
DATABASE_URL=your_neon_db_url
OPENROUTER_API_KEY=your_api_key
CLERK_SECRET_KEY=your_clerk_key
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_key
```

### 4. Run the development server

```bash
npm run dev
```
